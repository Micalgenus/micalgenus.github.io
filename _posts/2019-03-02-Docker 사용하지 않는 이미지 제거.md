Docker를 사용하다 보면 이미지가 계속 쌓이게 된다.

특히 예전에 사용하던 이미지들이 쌓이면서 용량이 부족해지는 경우가 생기는데, 아래 명령을 통해 지울 수 있다.

{% highlight bash %}
$ docker image prune -a
{% endhighlight %}

https://docs.docker.com/engine/reference/commandline/image_prune/
